# C# Coding Standards

- A `return` statement **must** be preceded by **exactly one blank line**
- Exception: If the `return` statement is the **only** statement inside its code block, no blank line is required
- No single line `if` statements are allowed - braces must be used
- No single line `else` statements are allowed - braces must be used
- Remove multiple blank lines - only one blank line is allowed
- Remove unused using statements
- Avoid trailing whitespace on any line
